# 🏆 Heart Disease Classification  

**Description:**  
This project focuses on building a machine learning model to classify the likelihood of heart disease using Python. The dataset includes key health indicators such as age, cholesterol levels, blood pressure, and more. The goal is to analyze patterns and improve predictive accuracy using multiple classification models.  

---

## 📊 **Models Used:**  
✔️ K-Nearest Neighbors (KNN)  
✔️ Logistic Regression  
✔️ Random Forest  
✔️ Principal Component Analysis (PCA)  

---

## 🚀 **Steps Completed:**  
1. Data Cleaning and Preprocessing  
2. Feature Scaling  
3. Model Training and Evaluation  
4. Performance Comparison Using Accuracy and F1 Score  

---

## 🎯 **Key Results:**  
✔️ Identified key health indicators influencing heart disease.  
✔️ Achieved high accuracy using Random Forest.  
✔️ PCA helped reduce dimensionality while maintaining performance.  

---

## 🛠️ **Skills Used:**  
✅ Python  
✅ Scikit-Learn  
✅ Pandas & NumPy  
✅ Data Preprocessing  
✅ Model Evaluation
